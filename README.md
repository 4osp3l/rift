## Rift

Rift is a lightweight, in-browser extension designed to help security researchers / bug bounty hunters easily explore recon data stored in ZIP files. It supports lazy-loading of large files, drag-and-drop of ZIPs, file previews, and shareable links for individual files.

## Features

> Drag & Drop ZIPs 
- Easily drop a ZIP archive into the app to load recon files.

> Lazy Loading for Large Files
- Preview massive files efficiently without freezing the browser.

> Open in Separate Window ( Chrome extension )
- Detach Rift into a standalone window for multitasking.

> Share Individual Files
- Generate a shareable link for a single file. Anyone opening the link can view the file in their browser.

## AI Assistant
Rift comes with a built-in AI Assistant that helps you analyze and understand the files you upload; you can also communicate with the AI directly.

## Installation
Chrome Extension ( Recommended)
1. Clone or download this repository.
2. Open chrome://extensions/ in your Chrome browser.
3. Enable Developer mode.
4. Click "Load unpacked" and select the Rift folder.
5. Pin the extension for quick access.

Standalone Web Version
1. Visit https://4osp3l.github.io/rift/ in your browser.
2. Drag & drop your ZIP file to get started.
